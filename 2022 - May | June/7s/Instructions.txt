7s - Instructions
Write a program to: • output each integer value between an inputted start number and an inputted end number, that ends with the digit 7. • output a final count of the number of values that are output.

For example:

The user enters 84 as the start number

The user enter 123 as the end number

The program should output, 87, 97, 107, 117, and a final count of 4.
